# Chat App 💬

A full-stack real-time chat app built with:

- ⚙️ Express.js & MongoDB (backend)
- ⚡ Vite + React (frontend)
- 🌐 Socket.IO (for real-time messaging)

## Features

- User authentication (signup/login)
- Real-time messaging
- Profile updates
- Image upload via Cloudinary

## Getting Started

```bash
# Backend
cd backend
npm install
npm start

# Frontend
cd frontend
npm install
npm run dev
